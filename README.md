# Two Skill Backend

## Contributing

1. Clone the repo
   ```sh
   git clone https://github.com/Kshubhranshu/cuvette-assignment.git
   ```
2. Installing dependencies
   ```sh
   npm i 
   npm run  install 
   ```
3. Start server
    ```sh
   [Node server]: npm run startApp
   [React ap]: npm run startServer 
   ```
